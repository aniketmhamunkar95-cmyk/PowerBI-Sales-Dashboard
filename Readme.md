 📊 Sales & Order Overview Dashboard (Power BI)
 🚀 Project Overview
This project is an interactive Sales & Order Overview Dashboard built using Power BI. It helps analyze business performance by tracking revenue, orders, and growth trends across different dimensions like time, region, and category.

---

 📌 Key Metrics
- 💰 Total Revenue: 15M
- 📦 Total Orders: 300
- 🛒 Average Order Value (AOV): 48.52K
- 📈 Year-over-Year Growth: 0.99%

---

 📊 Dashboard Features
- 📅 Monthly Revenue & Orders Trend Analysis
- 🌍 Region-wise Revenue Breakdown (North, West, East)
- 🧩 Category-wise Sales Distribution (Electronics, Furniture, Clothing)
- 📉 Growth Tracking (YoY %)
- 📌 KPI Cards for quick insights
- 🎛 Interactive filtering and dynamic visuals

---

🛠 Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization
